About

My project was inspired by my profession as a teacher. many a times we get overwhelmed by a need to balance our work and this has brought me to develop this application as a way of dealing with one of the challenges. as a student at Holberton School, i wanted to showcase my skills and create a portfolio projec that demonstrates my abilities in web development and user experience design

Features

Feature 1: Login page

This feature allows users to securely access their files and also protect information form unauthorised access

Feature 2: Student Profiles

This feature allows user to see the list of learners. it is designed in such a way that when a name is clicked to redirects the user to a full profile of a learner with all the learner's details including the results.

Feature 3: Student page

This feature allows the user to see individual learner results. It has a prev button which redirect the user the a previous page and a next button which redirects the user to the next page as listed on the student profile page
